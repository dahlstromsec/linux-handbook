# Git

Git is a distributed version control system used to track changes in files and collaborate on projects.

---

## git clone

### Purpose

Copies a remote repository to your local computer.

### Syntax

```bash
git clone URL
```

---

## git init

### Purpose

Creates a new Git repository.

### Syntax

```bash
git init
```

---

## git add

### Purpose

Stages changes for the next commit.

### Syntax

```bash
git add .
```

---

## git commit

### Purpose

Creates a snapshot of staged changes.

### Syntax

```bash
git commit -m "Message"
```

---

## git push

### Purpose

Uploads local commits to a remote repository.

### Syntax

```bash
git push
```

---

## git pull

### Purpose

Downloads and integrates remote changes.

### Syntax

```bash
git pull
```

---

## git branch

### Purpose

Lists or creates branches.

### Syntax

```bash
git branch
```

---

## git checkout

### Purpose

Switches branches or restores files.

### Syntax

```bash
git checkout BRANCH
```

---

## git switch

### Purpose

Switches between branches.

### Syntax

```bash
git switch BRANCH
```

---

## git merge

### Purpose

Combines one branch into another.

### Syntax

```bash
git merge BRANCH
```
